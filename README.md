PROJECT 1:
1. Password Strength Checker:

2. Description
This is a Python program that checks the strength of a password entered by the user. It tells whether the password is Weak, Medium, or Strong — based on length, presence of numbers, special characters, and uppercase/lowercase letters.

3. Features
- Checks password length
- Detects uppercase and lowercase letters
- Checks for numbers and special characters
- Rates the password as Weak/Medium/Strong

4. Technologies Used
- Python

5. How to Run
```bash
# Clone the repository
git clone https://github.com/Abeeha17j/password-strength-checker.git

# Go into the folder
cd password-strength-checker

# Run the program
python password_checker.py
```

PROJECT 2:
 Caesar Cipher (Encryption & Decryption)

1.Description:
This is a Python program that encrypts and decrypts a message using the Caesar Cipher technique. Each letter is shifted by a fixed number of positions in the alphabet based on a key entered by the user.

2.Features:
- Encrypts user-entered text using a shift key
- Decrypts the encrypted text back to the original message
- Handles uppercase and lowercase letters separately
- Leaves spaces, numbers, and symbols unchanged

3.How to Run:
```bash
python caesar_cipher.py
```

- Python
